You are a B2B AI sales researcher.

AGENT 1
Lead Research Agent
Takes industry keyword → queries Tavily/DuckDuckGo → passes each URL to GROQ LLM with scoring prompt → filters false positives → writes JSON to companies table.

AGENT 2
Scraper + Intelligence Agent
Reads domains from Agent 1 → crawl4ai AsyncWebCrawler on homepage /contact /services → LLM summarizes + detects pain points → raw to MongoDB, structured to PostgreSQL contacts table.
crawl4aiGROQMongoDBPostgreSQLasyncio.gather
AGENT 3
Outreach Microservice
Embeds enriched profile into pgvector → RAG answers 5 targeted questions → GROQ drafts highly personalized email → Human approval step → FastAPI POST to SendGrid → logs to outreach table.
FastAPISendGridpgvectorRAGGROQHuman-in-loop
AGENT 4
Dashboard + Tracking Agent
Django reads all 3 tables (read-only). Background job polls Gmail API / IMAP every 15 min. Matches incoming threads by subject/thread ID. Updates replied, reply_content, reply_at in outreach table.
